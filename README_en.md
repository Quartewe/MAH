## Features

- **Start Game**: Automatically start the game and wait for the main interface.
- **Dungeon Auto-Battle**:
  - Support farming for all permanent and event dungeons.
  - Customize challenge times, automatically use stamina items (Lanterns) to restore DP.
  - Intelligent support selection (Specific characters, AR, filter by stats/seeds for best support).
  - Flexible JSON script configuration for auto-battle and custom team building.
- **Stamina Auto-Battle**:
  - Support Chapter 14 flower stages and permanent daily quests.
  - Automatically use stamina potions by priority (Full -> Half -> Mini).
- **Claim Rewards**: Automatically claim weekly quests and gift box rewards.
