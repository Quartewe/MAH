## 功能列表

- **启动游戏**：自动启动游戏并等待进入主界面。
- **地城作战**：
  - 支持所有常驻地城及活动地城刷取。
  - 自定义挑战次数，自动使用灯油恢复 DP。
  - 智能助战选择（指定角色、AR、按数值/种子筛选最强助战）。
  - 支持灵活的 JSON 自动战斗脚本配置与自定义配队。
- **体力作战**：
  - 支持14章花本及常驻日常任务。
  - 自动按优先级（全->半->微）使用体力药。
- **领取奖励**：自动领取周常任务与礼物盒奖励。
