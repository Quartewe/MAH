## 功能列表

- **啟動遊戲**：自動啟動遊戲並等待進入主介面。
- **地城作戰**：
  - 支援所有常駐地城及活動地城刷取。
  - 自定義挑戰次數，自動使用燈油恢復 DP。
  - 智能助戰選擇（指定角色、AR、按數值/種子篩選最強助戰）。
  - 支援靈活的 JSON 自動戰鬥腳本設定與自定義配隊。
- **體力作戰**：
  - 支援14章花本及常駐日常任務。
  - 自動按優先級（全->半->微）使用體力藥。
- **領取獎勵**：自動領取周常任務與禮物盒獎勵。
